---
title: Get order status
description: "The polling fallback — GET /agent/v1/orders/{order_id} for reconciliation or when you can't accept webhooks."
---

```http
GET /agent/v1/orders/{order_id}
```

```json
{
  "order_id": "ord_8sd2k1n4",
  "status": "running",
  "instruction": "Buy a Yeti Tundra 45 hard cooler in tan at Amazon",
  "merchant": "amazon.com",
  "amount_charged": null,
  "order_number": null,
  "tracking": null,
  "created_at": "2026-06-10T17:04:22Z",
  "updated_at": "2026-06-10T17:08:10Z",
  "metadata": { "maive_order_id": "M-44821" }
}
```

Use this for reconciliation or if you cannot accept [webhooks](/api/webhooks).

<Warning>
  Poll no more often than once every 10 seconds per order.
</Warning>
